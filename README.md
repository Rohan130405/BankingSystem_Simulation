Banking System (Java + JDBC)

This project is a console-based banking application developed using Java and MySQL. It simulates basic banking operations and focuses on core backend logic along with database integration.

The application allows users to create an account with a PIN and a minimum balance. Users can deposit money, withdraw money, transfer funds to another account, check their balance, and view transaction history. Proper validations are implemented to ensure that incorrect PINs or invalid account numbers are handled safely.

This project uses core Java concepts such as object-oriented programming, exception handling, and JDBC for database connectivity. PreparedStatement is used to prevent SQL injection and ensure secure database operations.

To run the project, compile all Java files and include the MySQL connector jar in the classpath, then run the Main class.

Through this project, I learned how to design a backend system, integrate Java with a database, and handle real-world scenarios like transactions and validations.
